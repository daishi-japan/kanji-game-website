# 漢字学習ウェブサイト プランニング計画書

## 1. コンセプト (Concept)

### **「勉強を『攻略』に上書きする。」**

「勉強する」という行為そのものを「攻略する（ゲームを解く）」という行為にシステムごと書き換えます。「努力して覚える」のではなく、ソフトを**「インストール」するように、気づいたら頭に入っている状態**を目指します。

---

## 2. ターゲット分析 (Target Analysis)

### ターゲットA：長沼より（7歳・女の子）
*   **属性:** 小学校1年生。要領が良く、運動も勉強もポテンシャルが高い。
*   **現状:** 親の干渉により、勉強に対して「やらされている感」を持ち、少し嫌気が差している。
*   **願い:** 親にうるさく言われず、自分の好きなこと（ゲーム・収集）に没頭したい。

### ターゲットB：よりちゃんのお父さん・お母さん
*   **属性:** 教育熱心だが、子供との衝突を避けたい両親。
*   **現状:** 「宿題やりなさい」と言うことに疲れ、子供が勉強嫌いになることを懸念している。
*   **願い:** 子供には、嫌々やるのではなく、**前向きに楽しんで勉強してほしい**。その結果として、基礎学力が身につけば最高だと考えている。

## 3. バリュー・プロポジション・キャンバス (Fit Analysis)

### 【子供：よりちゃん】
| Customer Profile | Value Proposition |
| :--- | :--- |
| **Job:** 親にうるさく言われずに過ごしたい。自分の力で何かを成し遂げたい。<br>**Pains:** 親の監視、細かい指摘、強制される感覚。<br>**Gains:** 自由なプレイ体験、コンプリートの達成感、進化の驚き。 | **Product:** 漢字攻略型エンターテインメント（落ち物パズル）。<br>**Pain Relievers:** 親の介入を必要としないUI。「読み」というシンプルなルールで、誰にも邪魔されず没頭できる環境。<br>**Gain Creators:** 図鑑コンプリートやキャラ進化という明確な「攻略目標」。 |

### 【親：お父さん・お母さん】
| Customer Profile | Value Proposition |
| :--- | :--- |
| **Job:** 子供に自発的に学んでほしい。親子の時間を笑顔で過ごしたい。<br>**Pains:** 毎日の「宿題やった？」という確認がストレス。子供が勉強を嫌がる姿を見るのが辛い。<br>**Gains:** 子供が自発的に机に向かう姿。親子で「できたね！」と喜び合える関係。 | **Product:** 「ご褒美機能」付き学習ゲーム。<br>**Pain Relievers:** 子供が勝手に「攻略」し始めるため、監視や強制の必要がなくなる。<br>**Gain Creators:** 子供の「攻略成果」をご褒美で承認できる仕組み。子供の熱中が親の安心感に変わる。 |
