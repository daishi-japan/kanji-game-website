# Epic: リアルな書き取り攻略（メインモード）

## 概要
「読み」と対をなすメインコンテンツ。
デジタルの便利さ（書き順アニメーション、手本表示）と、アナログの学習効果（リアルなノートへの模写）を融合させ、確実に「書ける」ようにする。

## ユーザーストーリー

### 1. モード選択（よむ・かく）
*   **As a (誰):** 子供
*   **I want to (何):** ホーム画面で「よむ（おちもの）」か「かく（奥義）」か、今日の気分で遊び方を選びたい。
*   **So that (なぜ):** 読むだけでなく、書く練習もゲームとして楽しみたいから。

### 2. 書き順アニメーション（秘伝の動き）
*   **As a (誰):** 子供
*   **I want to (何):** 画面いっぱいに表示された漢字の「正しい書き順」をアニメーションで確認したい。
*   **So that (なぜ):** 親に聞かなくても、アニメーションを見て真似するだけで、正しい書き方を自律的にマスターできるから。

### 3. デジタルなぞり書き（アプリ内練習）
*   **As a (誰):** 子供
*   **I want to (何):** 画面上の手本を指でなぞって、書き順が合っているか判定してもらいたい。
*   **So that (なぜ):** ノートを用意するのが面倒な時でも、手軽に書き取りの練習ができるから。

### 4. リアルノートでの模写（データ出力）
*   **As a (誰):** 子供
*   **I want to (何):** 「本番モード」として、画面を見ながら自分のリアルなノートに漢字を書き写し、「書けた！」ボタンを押して完了したい。
*   **So that (なぜ):** 実際に紙に書くことで、「学校のテストでも絶対に書ける！」という自信を現実に持ち帰りたいから。

## 完了基準 (Acceptance Criteria)
*   [ ] ホーム画面から「書き取りモード」を開始できること。
*   [ ] 漢字の書き順アニメーションが再生できること。
*   [ ] 画面上で指を使ってなぞり書きができ、簡易的な正誤判定（書き順チェック）ができること。
*   [ ] 練習完了時に報酬（エサなど）が獲得できること。
